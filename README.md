# 1. 프로젝트 구조
- _data: 테마 커스터마이징하기 위한 데이터 파일이 위치함
- _include: 재사용 가능한 코드들, 공통 컴포넌트 코드들이 위치함
- _layouts: 페이지마다 디자인과 직접적으로 연결된 전체적인 레이아웃. 템플릿을 위한 코드를 한 곳에 보관할
- _posts: 블로그 포스트 파일이 위치함

# 2. 해야되는 설정
- 사이드 바 테마 및 설정 변경(스크롤 수정 등)
